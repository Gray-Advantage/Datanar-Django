Redirect API

This API allows you to manage redirects.

Endpoint: api/v1/redirects

GET /redirects

Retrieves a list of all redirects.

Response:

            [
                {
                    "id": 1,
                    "long_link": "http://example.com",
                    "short_link": "http://example.com/new",
                    "created_at": "2021-01-01T00:00:00Z",
                    "password": "",
                    "validity_clicks": 1,
                },
            ]

GET /redirects/{id}

Retrieves the details of a specific redirect.

Parameters:

  • {id}: the ID of the redirect

Response:

            {
                "id": 1,
                "long_link": "http://example.com",
                "short_link": "http://example.com/new",
                "created_at": "2021-01-01T00:00:00Z",
                "password": "",
                "validity_clicks": 1,
            }

DELETE /redirects/{id}

Deletes a specific redirect.

Parameters:

  • {id}: the ID of the redirect

Response:

Returns a 204 No Content response indicating that the redirect has been successfully deleted.
